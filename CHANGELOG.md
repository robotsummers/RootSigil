# Changelog

All notable changes to this project are documented in this file.

## v0.1.0 - 2026-02-19

Initial public release.

### Added
- Artifact scanning API with sync/async scan flows.
- Attestation issuance and verification endpoints.
- Revocation list publishing and signature support.
- Canonical manifest hashing and root hash derivation.
- Intake hardening for inline, zip, and git sources.
- Git intake safeguards for protocol, submodule, symlink, path, and size controls.
- API integration, contract, and OpenAPI conformance tests.
- CI workflow for lint, typecheck, and test.
